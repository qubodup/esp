3.4 Schleifen (Loops)
Die while-Schleife (whrend-loop) fhrt das Programm in dessen Rumpf/Krper aus, so lange
die Schleifen-/Laufbedingung y < 500 wahr ist.
1 x = 0
2
3 function love.draw()
4 y = 0
5
6 while y < 500 do
7 love.graphics.rectangle("fill",x,y,300,150)
8 y = y + 200
9 end
10 end
11
12 function love.mousepressed(mx, my)
13 x = x + 50
14 end

2. Verschiebe Zeile 4 in Zeile 2. Der Bildschirm sollte schwarz werden. Wieso?
